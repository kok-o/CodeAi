# CodeAI

**Бағдарламалауды үйренуге арналған және кодты интеллектуалды бағалайтын веб-қосымша**  
*Веб-платформа для обучения программированию с ИИ-репетитором и интеллектуальной оценкой кода.*

Это дипломная работа, представляющая собой инновационную образовательную платформу, созданную для помощи пользователям в освоении различных языков программирования с помощью персонального ИИ-наставника.

---

## О проекте / Жоба туралы

**Цель:** Создание удобной и современной платформы для обучения программированию, где искусственный интеллект выступает в роли личного репетитора. ИИ анализирует решения студентов, подсказывает лучшие подходы, выявляет ошибки и генерирует персонализированные учебные планы.

**Мақсаты:** Жасанды интеллект жеке репетитор рөлін атқаратын бағдарламалауды үйренуге арналған ыңғайлы және заманауи платформа құру. ЖИ студенттердің шешімдерін талдайды, ең жақсы тәсілдерді ұсынады, қателерді анықтайды және жеке оқу жоспарларын жасайды.

---

## Основной функционал / Негізгі мүмкіндіктер

- **Интерактивная IDE**: Написание и выполнение кода прямо в браузере с использованием движка Monaco Editor.
- **Поддержка множества языков**: Python, JavaScript, C++, Java, Go, Rust и другие.
- **ИИ-Репетитор (AI Mentor)**: Помогает находить ошибки, объясняет сложные концепции, но не решает задачу за пользователя, стимулируя его развивать собственные навыки.
- **Интеллектуальная оценка (Smart Evaluation)**: Автоматическая проверка кода с помощью изолированной среды (Sandbox), выполнение тестов и статический анализ.
- **Персональные планы обучения**: Нейросеть генерирует персональные треки обучения на основе текущей успеваемости студента (навыки, накопленный опыт, слабые стороны).
- **Система достижений и рейтингов**: Игровизация (Геймификация) процесса обучения с уровнями и рангами (от Новичка до Diamond I).
- **Ролевая система**: Личные кабинеты для разных типов пользователей: студентов, преподавателей (Teacher Studio) и администраторов (Admin Dashboard).

## Стек технологий / Технологиялар стегі

### Frontend (Клиенттік бөлім)
- **Фреймворк:** React.js + Vite
- **Анимации:** Framer Motion
- **Графики и аналитика:** Recharts
- **Редактор кода:** Monaco Editor

### Backend (Серверлік бөлім)
- **Среда:** Node.js
- **Фреймворк:** Express.js
- **ИИ-интеграция:** Stream API для работы с локальными или облачными LLM-моделями в режиме реального времени (например, интеграция с Ollama).
- **Выполнение кода:** Безопасное выполнение (Code Runner) пользовательского кода.

## Установка и запуск локально (Local Setup)

1. **Клонирование репозитория:**
   ```bash
   git clone <url-вашего-репозитория>
   cd diplom
   ```

2. **Установка зависимостей:**
   Вам нужно установить пакеты как для клиента, так и для сервера.
   ```bash
   npm install
   cd server
   npm install
   cd ..
   ```

3. **Настройка переменных окружения:**
   Создайте файл `.env` в корневой папке и в папке `server`, добавив необходимые порты, ключи и пути к базе данных.

4. **Запуск проекта:**
   Вы можете запустить клиентскую и серверную часть параллельно (если настроен соответствующий скрипт в `package.json`):
   ```bash
   npm run dev
   ```
   Или запустить их отдельно:
   - Клиент: `npm run dev`
   - Сервер: `cd server && npm start`

5. **Использование:**
   Откройте браузер по адресу `http://localhost:5173` для доступа к веб-интерфейсу платформы.
